# e-Commerce Modernizer REST POC

An example of clean architecture for an API, based on Spring Boot and Swagger

## Install

```
./gradlew assemble
```

## Test

```
./gradlew check
```

## Run

```
./gradlew bootRun
```

## Links

API documentation :
